## Instruções de Entrega do Desafio
Descrição do desafio: Você irá criar um relatório mais elaborado com base na sample financials do Power BI.

O arquivo de dados é: Financial_Sample.xlsx

Composição das colunas:
- (01) Segment.
- (02) Country.
- (03) Product.
- (04) Discount Band.
- (05) Units Sold.
- (06) Manufacturing Price.
- (07) Sale Price.
- (08) Gross Sale.
- (09) Discounts.
- (10) Sales.
- (11) COGS.
- (12) Profit.
- (13) Date.
- (14) Month number.
- (15) Month Name.
- (16) Year.

Fiquem atentos a: 
- (I) Estrutura definida. 
- (II) Botões de navegação que fornecem navegabilidade. 
- (III) Segmentadores utilizados e botões com imagem associado. 
- (IV) Utilize os indicadores e botões para selecionar diferentes visuais sobre um mesmo assunto. 
- (V) Objetos que definem o layout do relatório. 
- (VI) Gráficos (visuais) e os campos que os compõem. 
- (VII) Botões para navegabilidade. 
- (VIII) Segmentadores de dados. 

Lembre-se de: 
- Criar a segunda página do relatório.
- Publique o seu relatório no Power BI Service.
- Caso você tenha familiaridade fique livre para utilizar outros artifícios nos botões e outros.
- Submenta seu projeto através do link no github.

## Página 2: REPORT DE LUCRO DETALHADO.

### KPI Cards:
- Total Profit.
- Gross Margin %: ( Total Profit / Total Sales ) * 100
- Profit per Unit Sold.
- Profit Growth YoY %: crescimento percentual do lucro líquido em relação ao mesmo período do ano anterior.
- Quantidade de produtos.
- Qunatidade de Segmentos.

### Visual 1:
- Lucro por Produto.
- Gráfico de barras horizontais.
- Eixo X: Total Profit.
- Eixo Y: Product.

### Visual 2:
- Lucro por País / Segmento.
- Gráfico de barras empilhadas horizontais.
- Values: Total Profit.
- Axis: Country.
- Legend: Segment.

### Visual 3:
- Lucro ao longo do tempo.
- Gráfico de área.
- Values: Total Profit.
- Axis: Month name.
